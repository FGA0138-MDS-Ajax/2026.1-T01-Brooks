# Atas de reunião

As atas registram decisões, responsáveis, impedimentos e encaminhamentos importantes para o acompanhamento do projeto.

!!! warning "Registros pendentes"
    A equipe informou que as atas oficiais ainda precisam ser consolidadas. Esta página foi preparada para recebê-las sem criar informações que não tenham sido validadas pelos participantes.

## Informações recomendadas

Cada ata deve apresentar:

- data, horário e meio da reunião;
- participantes presentes e ausentes;
- pauta discutida;
- decisões tomadas;
- atividades distribuídas e responsáveis;
- impedimentos identificados;
- prazo e data da próxima reunião.

## Modelo

```markdown
# Ata de reunião — AAAA-MM-DD

## Participantes
- Nome dos participantes

## Pauta
- Assuntos discutidos

## Decisões
- Decisões aprovadas pela equipe

## Encaminhamentos
| Atividade | Responsável | Prazo |
|---|---|---|
| Descrição | Nome | Data |

## Impedimentos
- Impedimentos e riscos identificados
```

Quando os registros forem enviados pela gestão, cada reunião poderá receber uma página própria nesta seção.
